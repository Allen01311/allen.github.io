二、Flutter 介紹

Flutter是由Google開發的開源行動應用軟體開發套件， 用於Android、iOS、Windows、Mac、Linux的開發應用。 Flutter主要是採響應式框架而組成， Flutter框架提供了一組豐富的平台、布局和基礎組件。

作用
Dart 平台 在Windows、macOS和Linux上，Flutter在Dart虛擬機器中執行， 該虛擬機器具有即時編譯執行引擎。 在編寫和除錯時能夠即時編譯功能「Hot Reload」， 可以將對原始檔的修改注入正在執行的應用中。 在大多數情況下，對原始碼的更改可以立即在執行的應用中反映出來， 而無需重新編譯整份程式碼。
Flutter 引擎 主要使用C++開發，它實現了Flutter的核心庫， 包括動畫和圖形、檔案和網路I/O、 可存取性以及Dart執行環境和編譯工具鏈。
基礎庫 由Dart編寫，提供了用Flutter構建應用所需的基本的類和函式。
組件 Flutter是通過建立不同的組件完成使用者介面設計。 在Flutter中，一個組件代表使用者介面中不可變的一部分； 包括文字、形狀以及動畫在內的所有圖形都是用組件建立的。 而複雜的組件是由簡單的組件結合而成。
Flutter DevTools 用來分析、調試、診斷程式碼的狀態

三、Dart 介紹

Dart是由Google主導開發的開放原始碼程式語言， 類似JavaScript，Dart也是一種物件導向語言， 它在程式設計時強調物件（object）的類別（class）， 只允許單一繼承，語法風格與C語言接近。


四、Firebase 介紹

Firebase 是由 Google 提供的 CloudPlatform + Database 服務， 在租用 Server 的同時擁有 Firebase 的 NoSQL Database 支持， 同時 Google 也製作了一系列針對 Flutter 定制的 Firebase 套件， 使得 Flutter 和 Firebase 的聯動使用非常高效， 同時使用 Flutter 和 Firebase 讓 Query 語句和 View 界面能夠用 Dart 一種語言完成，極大提高了開發效率。 此外，一些列的 Google Ads、Realtime Database (Firestore)、 Firebase ML(Machine Learning) 等應用， Firebase Crashlytics、Firebase Performance 等監測後台都可以快速接入到 App 當中。
